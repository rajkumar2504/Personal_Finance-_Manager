# Personal Finance Manager

A comprehensive web application to manage your personal finances, track expenses, set budgets, and monitor savings goals.

## Features

- **Dashboard**: Quick overview of your financial health.
- **Transactions**: Track income and expenses with detailed categorization.
- **Budgets**: Set monthly limits for different categories and get visual progress alerts.
- **Goals**: Create savings goals and track your progress.
- **Reports**: Visualize spending habits with items and monthly trends.
- **Secure**: Data is stored locally on your browser (LocalStorage).

## Getting Started

1. Simply open `index.html` (or `login.html`) in your web browser.
2. Create a new account on the registration page.
3. Start adding your transactions!

## Tech Stack

- **HTML5** & **CSS3** (Custom Modern Design)
- **JavaScript (ES6+)**
- **Chart.js** for visualizations
- **LocalStorage** for data persistence

## File Structure

- `css/`: Stylesheets for layout, authentication, and responsive design.
- `js/`: Application logic separated by feature (auth, dashboard, expenses, etc.).
- `assets/`: Images and icons.
